# Решение задачи о p-медиане

Данный проект реализует решение задачи о p-медиане для выбора оптимального расположения заданного количества складов на железнодоржных станциях с целью минимизации транспортных затрат.

## Установка и запуск

Проект использует менеджер пакетов `uv` для управления зависимостями.

1.  **Установка зависимостей:**
    ```bash
    uv sync
    ```

2.  **Запуск программы:**
    ```bash
    python main.py
    ```

## Структура входных данных

Все входные файлы должны находиться в папке `input/` в корне проекта. Программа поддерживает файлы в формате Excel (`.xlsx`) и JSON.

### 1. Матрица смежности (`adjacency_matrix.xlsx`)

Файл содержит матрицу смежности для железнодоржных станций. Первая строка и первый столбец содержат названия станций. На пересечении строки и столбца указывается расстояние между ними.

**Пример формата:**

| Станции | ЦВЫЛЕВО | СИУЧ | МЫСЛИНО | ЛУЖСКАЯ | ПУСТЫНЬКА |
|:--------|:--------|:-----|:--------|:--------|:----------|
| ЦВЫЛЕВО | 0 | 10 | 15 | 20 | 25 |
| СИУЧ | 10 | 0 | 12 | 18 | 22 |
| МЫСЛИНО | 15 | 12 | 0 | 8 | 30 |
| ЛУЖСКАЯ | 20 | 18 | 8 | 0 | 16 |
| ПУСТЫНЬКА | 25 | 22 | 30 | 16 | 0 |

### 2. Вектор потребностей (`needs_vector.xlsx`)

Файл содержит объемы плетей (потребности) для каждой станции.

**Пример формата:**

| Станции | Объем плетей, км |
|:--------|:-----------------|
| ЦВЫЛЕВО | 17.359 |
| СИУЧ | 20.623 |
| МЫСЛИНО | 24.972 |
| ЛУЖСКАЯ | 6.6 |
| ПУСТЫНЬКА | 7.2 |

### 3. Параметры задачи (`input_p.json`)

JSON-файл, содержащий количество искомых медиан (p).

**Пример формата:**
```json
{
    "p_median_count": 2
}
```

## Логирование

Все промежуточные этапы работы программы и ее результат сохраняются в папку `logs/`. Файлы логов содержат временные метки для отслеживания истории запусков.
